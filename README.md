# HOME_work_Import.-Module.-Package

Домашнее задание по Python на тему модулей и пакетов.

Структура проекта:
- `main.py` — основной файл запуска;
- `application/salary.py` — модуль с функцией `calculate_salary()`;
- `application/db/people.py` — модуль с функцией `get_employees()`.

Запуск программы:
```bash
python main.py
```
